# NGSOC-Wazuh-SIEM

Welcome to the **NGSOC-Wazuh-SIEM** repository. This project integrates Next-Generation Security Operations Center (NGSOC) capabilities with the **Wazuh** security monitoring platform.

## Key Features
- **Centralized Security Monitoring**: Log analysis, integrity monitoring, and vulnerability detection.
- **Intrusion Detection**: Threat intelligence integration and custom rulesets for endpoint detection.
- **Incident Response**: Automated active responses to security incidents.

## Getting Started
Clone this repository and follow the configuration guides in the documentation.
